### 1.23
566. 重塑矩阵
> 暴力求解 AC
> 思路：首先判断是否合理（行列数相乘看是否相等），再将原矩阵全部展平为向量存储。
> 最后根据所求矩阵行列数，一行行填入答案。空间占用高，时间也高
>
> 官方解：通过两层索引，根据矩阵元素在矩阵中位置下标与列数的关系直接赋值：
> `ans[x/c][x%c]=nums[x/n][x%n]`;
> 时间空间均更优化
>
